Dom's Birthday Quest

Deploy instructions:
1. Upload the CONTENTS of this folder to GitHub, Netlify, or Vercel.
2. For GitHub Pages, make sure index.html, style.css, script.js, and the assets folder are at the repository root.
3. Passcode hint says birth date. Accepted codes: 2706, 270627, 0627, 27.

Files required:
- index.html
- style.css
- script.js
- aframe-day.png
- aframe-night.png
- bathroom.png
